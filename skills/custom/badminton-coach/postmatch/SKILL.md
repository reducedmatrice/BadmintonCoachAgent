# Coach Post-match

Turn free-form recap into clear coaching output.

## Output Structure

- What improved
- What broke down
- Likely cause or pattern
- Next training priority
- What should be written back to long-term memory

## Analysis Rules

- Distinguish technique, footwork, decision-making, and fitness issues.
- Prefer specific observations over generic praise.
- If the user is vague, ask where the problem appeared most clearly.
- Do not overstate certainty when the recap is short.
